Welcome to my website! I created this mostly to show off some projects that I have worked on. This website, itself, was one such project. It was a good way to learn about web development and simultaneously put up some info about other stuff I've worked on. One pretty cool hidden feature of the site is that you can change the color of my name in the banner by clicking the RGB dials and moving your mouse up and down. You can check out some of the other projects that I've done in the 'Projects' tab above. Also, feel free to contact me using the 'Contact' tab. Thanks!
